# footkey
A foot switch keyboard using Seeeduino XIAO.

本体写真

Seeed StudioのSeeeduino XIAOを使って作られたフットスイッチのキーボードプロジェクトです。
YAMAHAのフットスイッチFC5を最大４台接続し、USBでパソコンへ接続するとショートカットキーに割り当てることができます。
手で操作することが困難な状況でも、足で操作することが可能です。
